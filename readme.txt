Если у вас не установлена PyOpenGL, то придётся установить
следующим образом, потому как, скорее всего, у вас официальная
установка не сработает должным образом.
Для установки PyOpenGL:
1) файлы PyOpenGL_accelerate-3.1.5-cp37-cp37m-win_amd64.whl
 и PyOpenGL-3.1.5-cp37-cp37m-win_amd64.whl переместить в
директорию Python\Scripts (скорее всего это C:\Program Files\Python\Scripts),
в этой директории должен быть установщик pip.exe и/или pip3.exe;
2) из командной строки установить, используя установщик pip3 (pip),
библиотеку PyOpenGL, введя команды 
pip3 install PyOpenGL-3.1.5-cp37-cp37m-win_amd64.whl
pip3 install PyOpenGL_accelerate-3.1.5-cp37-cp37m-win_amd64.whl
3) дождаться завершения установки, готово).
То есть, по сути, это установка нелицензионной библиотеки.
Толком так и не понял, почему официальную библиотеку до конца не доделали.
